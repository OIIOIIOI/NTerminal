NTerminal
=========

Basic console for the Nuggeta API

Available commands:
- start [APP_ID]
- getGames
- createGame
- stopGame [GAME_ID]
- stopAllGames
- help

OpenFL project, but only tested for the Flash platform.
You'll need to copy the Nuggeta API files in the src-nuggeta folder.

Demo here: http://01101101.fr/NTerminal/
